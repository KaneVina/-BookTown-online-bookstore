## 📖 Overview

**BookTown** is a full-stack online bookstore system designed to modernize book retail — moving from manual order-taking via Facebook/Zalo to a fully automated, 24/7 web platform. Customers can browse, search, purchase, and review books seamlessly, while admins manage inventory, orders, and revenue analytics from a dedicated dashboard.

---

## 🎯 Business Objectives

| Goal | Description |
|------|-------------|
| 📈 **Revenue Growth** | Expand sales through a 24/7 accessible online channel |
| ⚡ **Automation** | Reduce manual order handling by **30–40%** |
| 📦 **Inventory Control** | Real-time stock updates on every order |
| 🤝 **Customer Trust** | Transparent product info, review system, and order tracking |
| 📊 **Analytics** | Revenue dashboards by day / week / month |

## 🔄 Business Process

### As-Is (Current)
```
Customer → Contacts via Facebook/Zalo/Phone
         → Admin records order manually (Excel/notebook)
         → High error rate: wrong orders, stock gaps, missed customers
         → No real-time inventory visibility
```

### To-Be (With BookTown)
```
Customer → Self-orders on website (24/7)
         → Order auto-recorded → Inventory auto-updated
         → Admin views live dashboard: orders, stock, revenue
         → Zero manual data entry needed
```
